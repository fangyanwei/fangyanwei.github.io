you must restart adb and eclipse

发表于 2018-07-22 | 分类于 Android
字数统计: 375 | 阅读时长 ≈ 1

前阵子主板坏了，系统重新安装过，导致很多环境都要重新配置。今天想写个android测试程序，运行的时候提示：you must restart adb and eclipse，然后重启了这两货还是不行。

百度了下，有下面这些解决方案：
方法一：

之后有人说将tools和platform-tools的路径加到path 还有classpath里面，但是好像没有用

方法二：

adb kill-server

adb start-server

方法三：

直接重启电脑，感觉这个才是最管用，那天就是一下子开了好几个模拟器，之后eclipse就犯二了！

方法四：

一种情况是“豌豆荚”或“手机助手”类软件占用了adb,在任务管理器中停止“豌豆荚”以及adb后重启adb后重启eclipse

注意把任务管理器中的豌豆荚关闭了！

Base64编码及其作用

发表于 2018-06-16 | 分类于 C++
字数统计: 879 | 阅读时长 ≈ 3

最重要的是

1.便于网络传输。

2.不可见性。

（一）Encoding VS. Encryption

很多人都以为编码（Encoding）和加密（Encryption）是同一个意思。编码和加密都是对格式的一种转换，但是它们是有区别的。编码是 公开的，比如下面要介绍的Base 64编码，任何人都可以解码；而加密则相反，你只希望自己或者特定的人才可以对内容进行解密。

（二）Base 64编码

Base 64 Encoding有什么用？举个简单的例子，你使用SMTP协议 （Simple Mail Transfer Protocol 简单邮件传输协议）来发送邮件。因为这个协议是基于文本的协议，所以如果邮件中包含一幅图片，我们知道图片的存储格式是二进制数据（binary data），而非文本格式，我们必须将二进制的数据编码成文本格式，这时候Base 64 Encoding就派上用场了。

我是三级标题

第二种写法：
